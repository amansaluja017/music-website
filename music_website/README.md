## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn or bun

### Installation

1. Clone the repository:
```bash
git clone https://github.com/amansaluja017/music-website.git
```

2. Install dependencies:
```bash
npm install
# or
yarn install
# or
bun install
```

3. Run the development server:
```bash
npm run dev
# or
yarn dev
# or
bun dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser
